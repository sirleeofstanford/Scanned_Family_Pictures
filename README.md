# Scanned Family Pictures

A collection of family photographs that have been digitally scanned and preserved.  
This repository is intended to share memories with family members and keep our history safe.

## Purpose
- Preserve old family photos in digital format
- Make them easily accessible to relatives
- Provide a central place for organizing and sharing

## How to Access
- Browse the repository to view available photos
- Download images directly if you’d like to keep a copy
- Feel free to create folders for specific events, years, or family branches

## Contributing
Family members are welcome to:
- Upload additional scanned photos
- Add descriptions or notes to images (e.g., names, dates, locations)
- Help organize photos into meaningful categories

## Guidelines
- Please respect privacy: only upload photos you’re comfortable sharing
- Use clear filenames (e.g., `Grandma_Birthday_1985.jpg`)
- Avoid duplicates when possible

## License
This repository is for **family use only**.  
Photos are not to be redistributed outside of our family circle.
